# 업체 협업 업무관리 서비스 기획서 (MVP)

## 1. 서비스 개요

다수의 업체와 협업 시 발생하는
업무, 일정, 담당자, 진행 상태 등을
쉽고 직관적으로 공유하고 관리할 수 있는 협업 서비스.

복잡한 ERP가 아닌:

“엑셀보다 쉽고, 카톡보다 안 잊어버리는 협업 서비스”

를 목표로 한다.

---

# 2. 서비스 목적

현재 업체 간 협업은:

- 카카오톡
- 전화
- 엑셀
- 메모
- 개인 캘린더

등으로 분산되어 관리되는 경우가 많다.

이로 인해:

- 업무 누락
- 일정 혼선
- 담당자 혼동
- 진행 상태 확인 어려움

문제가 자주 발생한다.

본 서비스는 이를 단순하고 직관적으로 해결하는 것을 목표로 한다.

---

# 3. 핵심 기능

## 3-1. 업체 관리

협업 중인 업체 정보를 등록 및 관리.

### 등록 항목

| 항목 | 설명 |
|---|---|
| 업체명 | 업체 이름 |
| 업체 담당자 | 해당 업체의 대표 연락 담당자 |
| 연락처 | 전화번호 |
| 이메일 | 메일 주소 |
| 메모 | 참고 사항 |

---

## 3-2. 업무 관리

업체별 업무를 등록하고 진행 상태를 관리.

### 등록 항목

| 항목 | 설명 |
|---|---|
| 업무명 | 진행 업무 |
| 관련 업체 | 연결된 업체 |
| 업무 담당자 | 실제 업무 수행 담당자 |
| 시작일 | 업무 시작 |
| 마감일 | 완료 예정일 |
| 상태 | 대기 / 진행중 / 확인요청 / 완료 |
| 메모 | 업무 내용 |

---

## 3-3. 일정 관리

업무 일정을 캘린더 형태로 제공.

### 기능

- 오늘 업무 보기
- 마감 예정 업무
- 업체별 일정 필터
- 월간 캘린더 보기

---

## 3-4. 댓글 및 진행 공유

업무별 간단한 커뮤니케이션 제공.

예시:

시안 전달 완료
검수 요청드립니다
수정본 업로드 예정

---

## 3-5. 알림 기능

기본적인 업무 알림 제공.

### 알림 종류

- 마감 하루 전 알림
- 상태 변경 알림
- 댓글 등록 알림

---

# 4. 사용자 구조

## 업체 담당자

- 외부 업체의 대표 연락 담당자
- 업체 단위 정보

예:

A업체 - 김부장
B업체 - 이과장

---

## 업무 담당자

- 실제 업무를 수행하는 담당자
- 업무별로 변경 가능

예:

상품 등록 - 홍길동
배너 제작 - 김철수

---

# 5. 화면 구성

## 1. 대시보드

- 오늘 마감 업무
- 진행중 업무
- 완료 업무
- 최근 댓글

---

## 2. 업체 관리

업체 등록 및 조회.

---

## 3. 업무 관리

칸반 또는 리스트 형태.

대기 → 진행중 → 확인요청 → 완료

---

## 4. 캘린더

업무 일정 시각화.

---

# 6. 차별화 포인트

기존 협업툴은:

- 기능이 너무 많고

- 사용이 어려운 경우가 많다.

- 쉬운 공유
- 직관적인 상태 관리

에 집중한다.
